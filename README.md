PawVerse — веб-платформа для ухода за питомцами: записи к услугам, AI-анализ фото/видео, GPS-трекинг и Telegram-бот.

## О проекте
- Backend: Node.js + Express + Prisma + PostgreSQL
- Frontend: HTML/CSS/JS (статические файлы в `front/`)
- AI: Python + Ultralytics YOLO (`paw_pal_clean/paw_pal/`)

## Основные возможности
- Личный кабинет владельца и партнёра
- Управление питомцами (включая аватары и GPS)
- Онлайн-запись к услугам
- AI-диагностика по фото/видео через YOLO
- Отображение питомцев на карте

## Структура
- `backend/` — API, Prisma, интеграции
- `front/` — клиентская часть
- `paw_pal_clean/paw_pal/` — Python-скрипты обучения/инференса YOLO
- `runs/` — результаты тренировки/веса модели
